# Campaign Stats

`GET /v1/campaigns/:id/stats`

Returns the campaign with live counters. Same payload as `GET /v1/campaigns/:id`.

```bash
curl https://api.inboundsage.com/v1/campaigns/CAMPAIGN_ID/stats \
  -H "Authorization: Bearer is_live_YOUR_KEY"
```
