# Activate Campaign

`POST /v1/campaigns/:id/activate`

Start sending a `DRAFT` or `PAUSED` campaign.
If `scheduledAt` is set and in the future, status becomes `SCHEDULED`.
Otherwise it becomes `RUNNING` immediately.

```bash
curl -X POST https://api.inboundsage.com/v1/campaigns/CAMPAIGN_ID/activate \
  -H "Authorization: Bearer is_live_YOUR_KEY"
```

```json Response
{
  "id": "5832df51-...",
  "status": "RUNNING",
  "started_at": "2026-05-25T02:25:29.000Z"
}
```

Emits `campaign.started` to subscribed webhooks. Idempotency does not apply to activate — only to [create](/api-reference/campaigns/create).

## Other campaign actions

| Endpoint | Description |
|----------|-------------|
| `POST /v1/campaigns/:id/pause` | Pause a running campaign |
| `POST /v1/campaigns/:id/cancel` | Cancel permanently |
